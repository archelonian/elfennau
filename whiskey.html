Whiskey /

Interactive Feature Switchboard

 

Whiskey is an interactive feature switchboard. Use the switchboard on the left-hand side to select a set of phonological features to see which segments have those features; or, select a segment on the right-hand to see which phonological features it has.

v0.2 (13 January 2018) Added „reset“ button and „hide“ button; added feature that greys out segments which lack an inapplicable feature (e.g. labials are greyed out if [−anterior] is selected); added table-to-switchboard communication.

v0.1 (12 January 2018) Initial release.

Whence comes the source of this information?

The information is drawn from the 2011 edition of Jason Riggle's phonological features chart, with supplementary information drawn from this website that I found.

Why aren't the inputs sanitised to prevent contradictory features (for example, [−labial] and [+rounded])?

I was going to, but it would take too much effort, so it's somewhat reliant on the user to not be a buffoon.

Is this application still in development?

Until I get lazy or forget about this, I'm planning to add vowel support, table-to-switchboard communication, and meta-documentation about each individual feature and segment.

Why is the name of this application „Whiskey“?

The English word „whiskey“ derives from the Irish word „uisce“ /ɪʃcə/, meaning „water“. Varieties of water include the Oscan S.Pellegrino sparkling natural mineral water, whose only blemish is its ownership by the dreadful company Nestlé.
